Lesson 1

Composition of Functions

The output of one function is the input of another.

expressed as:

(f ∘ g) = f(g(x))

read as:

"f of g of x" or "f composition of g of x"

Example:

1. f(x) = 3x + 1
    g(x) = 2x + 1

(f ∘ g) = f(g(x))
= 3(2x + 1) + 1
= 6x + 3 + 1
= 6x + 4

(g ∘ f) = g(f(x))
= 2(3x + 1) + 1
= 6x + 2 + 1
= 6x + 3

(f ∘ f) = f(f(x))
= 3(3x + 1) + 1
= 9x + 3 + 1
= 9x + 4

2. f(x) = x2 + 4x + 4
    g(x) = x + 1

(f ∘ g)(x) = f(g(x))
= (x + 1)2 + 4(x + 1) + 4
= (x + 1)(x + 1) + 4(x + 1) + 4
= (x2 + 2x + 1) + (4x + 4) + 4
= x2 + 6x + 9
